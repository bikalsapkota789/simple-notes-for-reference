Basic working structures

start a new project

django-admin.py startproject <project-name> (This command must be written within your project directory)

start a project in virtual environment

virtualenv projectname-env projectname-env/bin/activate

start a new app

python manage.py startapp <app-name> (put your in INSTALLED_APPS list within /settings.py file)

create superuser(i.e. admin)

python manage.py createsuperuser

run server

python manage.py runserver

migrations

python manage.py migrate python manage.py makemigrations (do everytime after updating models.py from within your directory)

|browser| --> |server| --> |urls.py| --> |views.py| --> |templates.html|
